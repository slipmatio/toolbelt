# Slipmat Toolbelt

General utilities for Web development

## Features

### @slipmatio/toolbelt/browser

- `storageAvailable(type: 'localStorage' | 'sessionStorage'): boolean`
## Installation

`yarn add @slipmatio/toolbelt`
## Development

### Install dependencies

`yarn`

### Run development server

`yarn dev`

### Run tests

`yarn test`
